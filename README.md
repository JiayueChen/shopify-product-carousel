# Product Carousel for Shopify
This is a Shopify implementation of the original carousel design from [carousel-challenge](https://github.com/JiayueChen/carousel-challenge).

Preview Link: https://test-store-1100000000000000000000000000000003180.myshopify.com/


## Features
- Customizable section settings for title, product limit, and collection selection
- Product badges (e.g., "Best Seller") and discount badges (e.g., "Save 20%") pulled from product metafields
- Dynamic star rating icons generated from metafield rating values
- Review count displayed from product metafields
- Responsive design with mobile show more/less functionality


## Note
- Add the section directly to your Shopify theme.
- This section uses `!important` declarations to prevent style conflicts with theme CSS. Some styles may still not work depending on theme's CSS specificity.


## Known Issues
- Scrollbar styling not fully optimized for all Shopify theme variations
